# Christoph Willibald Gluck — Le calme rentre dans mon coeur

From _Iphigénie en Tauride_ (1779). This score is derived from [Paris: Des Lauriers, 1780](<https://imslp.org/wiki/Iphig%C3%A9nie_en_Tauride,_Wq.46_(Gluck,_Christoph_Willibald)>).

Note that the rhythm in the violas differs from German editions.
The first bars and final bar are editorial suggestions, delimited by double barlines.
